Simplistic glTF online viewer

Note

This glTF viewer is very limited in its capabilities and does not fully support glTF

It is basically an educational starting point to parsing glTF and only offers rudimentary display capabilities

Limitations

  • Assumes triangle type for all primitives
  • Ignores techniques, shaders, materials and displays basic wireframes
  • Does not support animations
  • Does not support camera information
  • Loads the default scene
  • Ignores matrix / TRS information
  • Requires indices

Try it

Pick the .gltf and all .bin files to display below and click "Load files"
